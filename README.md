# 📒 Introduce

이 프로젝트는 42SEOUL 에서 진행한 팀프로젝트 이며 HTTP/1.1기준을 적용한 Customized 한 웹서버를 만드는 과제 입니다.
실제 Nginx와 동작방식을 비교해가며 프로젝트를 진행 하였으며 HTTP/1.1을 기준으로 하고있지만 select함수를 사용한
multiplexing이 적용되어 있습니다.
